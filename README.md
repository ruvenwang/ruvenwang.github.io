To run this project on your local machine:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/ruvenwang/ruvenwang.github.io.git
    cd ruvenwang.github.io
    ```

    If you have Python installed:

    ```bash
    npx http-server .
    ```

3.  **Open in Browser:**
    Navigate to `http://localhost:8000`
